# q6 tier: a 6-bit weight repack for quality-first users

2026-07-12. Ask: a bigger quant for people who want a 6-bit model.
Constraint: no new kernels -- the engine speaks Q4_G64 and Q8_G128; the
policy space is which tensors ride Q8.

## Design (decided by the P0.5 sensitivity study, 2026-07-02)

q6 = v1.4 + ffn_down ALL promoted to Q8:

    tools/repack.py BF16.gguf out.q27 --q8 '(ssm_out|attn_output|ffn_down)\.' --tag q6-v1

- ffn_down ALL is the single best measured quality lever left: +2.76 GB
  for 29% of the remaining gap to Q5_K_M (P0.5 ceiling probe, PPL
  7.2139 -> 7.1466 on the qwopus corpus scale). Sensitivity is spread
  uniformly across layers -- there is no cheap subset, so the tier takes
  all of it.
- GDN in-projections stay Q4 ON PURPOSE: promoting them measured WORSE
  (+0.36% PPL; their Q4 errors partially cancel downstream).
- ffn_gate/up promotion is unmeasured and ~+5.5 GB more; that is an
  8-bit-class artifact, out of scope for this tier.
- Landing: ~20.5 GB file = 6.0 bits/param; ~18.6 GB streamed per decode
  round (was 15.8).

## Gates

1. NLL A/B, identical flags (--nll full corpus, fp16 KV, --nll-chunk
   2048), v1.4 vanilla vs q6 vanilla. Expect q6 below v1.4 by roughly
   the study delta; any regression kills the ship.
2. Q5_K_M bar on the same corpus (llama-perplexity) for the model-card
   claim.
3. New canonical: q6 is a new artifact -- record its own md5 of the
   canonical --tokens run (both CLI fp16 and the fp8/turbo3 serving
   paths must be deterministic per existing gates; no cross-artifact
   bitwise claim).
4. Speed price, measured not estimated: short-bench 5-prompt suite +
   26K cctx server replay vs the v1.4 numbers. Expect ~-10-15% decode
   (bytes 15.8 -> 18.6/round); acceptance coupling may claw some back
   (v1.4's promotion RAISED tok/rnd).
5. Serving envelope: 5090 auto-ctx caps for fp8 + turbo3; explicit 3090
   verdict (expected: does NOT fit -- 20.5 GB + fixed stack > 24 GB;
   the 5.25 bpw artifact remains the 24GB-card answer).

## Ship

HF: second .q27 in signalnine/Qwen3.6-27B-MTP-q27 + CHECKSUMS.md5 line
+ model-card section (what it is, who it's for, measured PPL/speed/ctx
deltas vs the 5.25 bpw default). README: one row in the performance
model + a line in Quickstart model download. BUILDLOG + memory.

## Non-goals

Q6 kernel (new dtype), ffn_gate/up promotion (8-bit-class), AWQ-style
importance-weighted scales (the documented path if quality demands ever
exceed what uniform promotion buys), Qwopus q6 (vanilla is the public
standard; qwopus-q6 only if asked).
